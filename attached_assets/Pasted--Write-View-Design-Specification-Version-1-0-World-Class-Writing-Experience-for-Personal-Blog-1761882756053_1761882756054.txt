# Write View Design Specification
**Version 1.0** | World-Class Writing Experience for Personal Blogging

---

## üéØ Design Philosophy

### Core Principles

1. **Words Come First**
   - Writing experience is 1000x better than typical editors
   - Large, readable text (22px/1.375rem) for older adult users
   - Distraction-free interface that disappears when not needed
   - Typography that makes typing a joy

2. **AI as Assistant, Not Hero**
   - AI tools live in supportive sidebar, not center stage
   - Workflow prioritizes raw writing ‚Üí AI enhancement ‚Üí publishing
   - Functions are helpful but never dominant
   - Manual save (Cmd+S) more prominent than AI features

3. **Elegant Minimalism**
   - Clean, spacious interface
   - Generous whitespace
   - Subtle animations that enhance without distracting
   - Focus mode for pure writing immersion

4. **Thoughtful User Experience**
   - Auto-save functionality (peace of mind)
   - Auto-focus on content area (start writing immediately)
   - Keyboard shortcuts for power users
   - Real-time statistics without overwhelm

## üé≠ Interaction Patterns

### Auto-Save System

**Trigger:**
- Fires 2 seconds after user stops typing
- Debounced to prevent excessive saves
- Only saves if title OR content exists

**Implementation:**
```javascript
useEffect(() => {
  if (!title && !content) return;
  
  const timer = setTimeout(() => {
    setLastSaved(new Date());
    localStorage.setItem('draft', JSON.stringify({ title, content }));
  }, 2000);

  return () => clearTimeout(timer);
}, [title, content]);
```

**Visual Feedback:**
- Top bar shows: "‚úì Saved [time]"
- Check icon (w-3 h-3)
- Relative time updates in real-time
- Manual Cmd+S shows toast: "Saved"

### Draft Persistence

**On Mount:**
```javascript
useEffect(() => {
  const saved = localStorage.getItem('draft');
  if (saved) {
    const draft = JSON.parse(saved);
    setTitle(draft.title || "");
    setContent(draft.content || "");
  }
}, []);
```

**On Publish:**
```javascript
// After successful publish
setTimeout(() => {
  setTitle("");
  setContent("");
  localStorage.removeItem('draft');
}, 500);
```

### Real-Time Statistics

**Word Count:**
```javascript
const words = content.trim().split(/\s+/).filter(word => word.length > 0);
setWordCount(words.length);
```

**Character Count:**
```javascript
setCharCount(content.length);
```

**Reading Time:**
```javascript
// 200 words per minute average
setReadTime(Math.ceil(words.length / 200));
```

**Typing Indicator:**
```javascript
// Shows user is actively typing
setIsTyping(true);
clearTimeout(typingTimerRef.current);
typingTimerRef.current = setTimeout(() => {
  setIsTyping(false);
}, 1000);
```

### Keyboard Shortcuts

**Escape:**
- Exits focus mode (if active)
- Single press only

**Cmd/Ctrl + Shift + F:**
- Toggles focus mode on/off
- Prevents default browser behavior

**Cmd/Ctrl + Shift + A:**
- Opens/closes AI sidebar
- Prevents default browser behavior

**Cmd/Ctrl + S:**
- Manual save trigger
- Shows success toast
- Prevents browser save dialog

**Implementation:**
```javascript
useEffect(() => {
  const handleKeyDown = (e: KeyboardEvent) => {
    if (e.key === 'Escape' && focusMode) {
      setFocusMode(false);
    }
    
    if ((e.metaKey || e.ctrlKey) && e.shiftKey && e.key === 'f') {
      e.preventDefault();
      setFocusMode(!focusMode);
    }
    
    // Additional shortcuts...
  };

  window.addEventListener('keydown', handleKeyDown);
  return () => window.removeEventListener('keydown', handleKeyDown);
}, [focusMode, showAI]);
```

### AI Transformation Flow

**User Actions:**
1. Writes content in textarea
2. Opens AI sidebar (if not open)
3. Clicks transformation button
4. Sees loading state
5. Receives transformed content (simulated)

**Visual Feedback:**
```javascript
// Loading state
toast.loading(`Applying ${type}...`);
setIsTransforming(true);

// Button shows loading
<Button disabled={isTransforming}>
  <Sparkles className={isTransforming ? 'animate-pulse' : ''} />
  {isTransforming ? 'Transforming...' : 'Transform My Words'}
</Button>

// Success state
setTimeout(() => {
  setIsTransforming(false);
  toast.dismiss();
  toast.success(`${type} transformation complete!`);
}, 1500);
```

**Disabled States:**
- All AI buttons disabled when content is empty
- Visual: opacity-50, cursor-not-allowed
- Clicking shows toast: "Please write something first"

### Publishing Flow

**Validation:**
```javascript
if (!title.trim()) {
  toast.error("Please add a title before publishing");
  return;
}
if (!content.trim()) {
  toast.error("Please write some content before publishing");
  return;
}
```

**Success Flow:**
1. Loading toast: "Publishing your words..."
2. 1-second delay (simulated API call)
3. Success toast: "Published "Title" ‚Äî 234 words preserved"
4. 500ms delay
5. Clear form and localStorage
6. Ready for next post

### Clear Functionality

**Behavior:**
- Only active if title OR content exists
- Shows confirmation dialog: "Are you sure you want to clear your writing?"
- On confirm: Clears all fields, removes draft, shows toast
- On cancel: No action

### Key Functions

**Format Time:**
```typescript
function formatTime(date: Date): string {
  const now = new Date();
  const diffMs = now.getTime() - date.getTime();
  const diffMins = Math.floor(diffMs / 60000);
  
  if (diffMins < 1) return 'just now';
  if (diffMins === 1) return '1 min ago';
  if (diffMins < 60) return `${diffMins} mins ago`;
  
  return date.toLocaleTimeString('en-US', { 
    hour: 'numeric', 
    minute: '2-digit',
    hour12: true 
  });
}
```

## üìä User Experience Flows

### First-Time User Journey

1. **Enters Write View**
   - Sees welcome message with ornament
   - Encouraging headline appears
   - Cursor auto-focuses in content area after 500ms

2. **Starts Writing**
   - Types naturally (no structure required)
   - Welcome message fades out
   - Word count begins updating in real-time
   - Auto-save starts working after 2-second pause

3. **Explores AI Tools** (Optional)
   - Clicks "AI Assistant" button
   - Sidebar slides in from right
   - Sees transformation options
   - Can apply or continue writing

4. **Uses Focus Mode** (Optional)
   - Clicks "Focus Mode" or uses Cmd+Shift+F
   - UI fades away
   - Text enlarges for immersive writing
   - Presses Esc when ready to return

5. **Publishes**
   - Clicks "Publish" button
   - Sees loading state
   - Gets success confirmation
   - Form clears for next post

### Experienced User Journey

1. **Returns to Write View**
   - Previous draft automatically loads
   - Cursor focuses immediately
   - Continues where left off

2. **Uses Keyboard Shortcuts**
   - Cmd+Shift+F for focus mode
   - Cmd+Shift+A for AI sidebar
   - Cmd+S for manual save
   - Faster, more efficient workflow

3. **Rapid Writing Workflow**
   - Writes freely without structure
   - Uses AI transform when ready
   - Publishes quickly
   - Clears for next post

### Error Recovery

**Lost Content:**
- Auto-save every 2 seconds prevents loss
- Draft persists across sessions
- Manual save option (Cmd+S)

**Accidental Clear:**
- Confirmation dialog prevents accidents

## üé® Special Features

### 1. Auto-Focus on Content

**Purpose:** Remove friction from starting to write

**Implementation:**
```javascript
setTimeout(() => {
  if (contentRef.current) {
    contentRef.current.focus();
  }
}, 500);
```

**Delay:** 500ms (allows page to settle, prevents jarring)

### 2. Bottom Gradient

**Purpose:** Visual indicator of infinite scroll space

**Implementation:**
```tsx
<div className="absolute bottom-0 left-0 right-0 h-24 
                bg-gradient-to-t from-background to-transparent 
                pointer-events-none" 
/>
```

**Effect:** Subtle fade from background color to transparent

### 3. Dynamic Editor Width

**Purpose:** Optimize reading width based on sidebar visibility

**Implementation:**
```tsx
<motion.div 
  layout
  className={`flex-1 transition-all duration-300 ${
    showAI ? 'max-w-3xl' : 'max-w-4xl mx-auto'
  }`}
>
```

**Result:** Editor centers when sidebar closed, shifts when open

### 4. Typing Indicator

**Purpose:** Show active typing state for future features

**Implementation:**
```javascript
// Set typing to true on change
setIsTyping(true);

// Clear after 1 second of no typing
clearTimeout(typingTimerRef.current);
typingTimerRef.current = setTimeout(() => {
  setIsTyping(false);
}, 1000);
```

**Future Use:** Could drive auto-save indicator, collaboration features

### 5. Relative Time Formatting

**Purpose:** Human-friendly save timestamps

**Formats:**
- "just now" (< 1 minute)
- "2 mins ago" (1-59 minutes)
- "3:45 PM" (60+ minutes)

**Updates:** Real-time via Date object in state
